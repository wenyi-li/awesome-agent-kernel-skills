## Description

When `num_postprocess_workers > 0` (a typical configuration for disaggregated serving), responses arrive via the `PostprocWorker.Output` branch of `_handle_response()`, which bypassed the `do_tracing()` call entirely. As a result, no llm_request OTLP spans were emitted for these requests.
