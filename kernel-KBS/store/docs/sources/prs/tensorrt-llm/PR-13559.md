## Release Notes

* **New Features**
  * Added support for Laguna language models with MoE and specialized attention mechanisms.
  * Introduced NVFP4 quantization enabling 4-bit float precision.
  * Added FP8 block-scale MoE inference optimization for improved performance.
  * Enabled KV-cache FP8 quantization support.
  * Added reasoning parser support for Laguna models.

* **Improvements**
  * Enhanced quantization configuration handling with improved safety checks.
  * Extended module exclusion patterns to support regex expressions alongside glob patterns.




## Description

Adds TensorRT-LLM PyTorch-backend support for **Poolside's Laguna model family**, including the current generation **Laguna-XS.2** in bf16, FP8 block-scale, and NVFP4 variants.

Laguna-specific bits: per-head softplus output gate (`g_proj` pre-`o_proj`), per-layer attention-head counts, per-layer dual RoPE (yarn for full-attention, default linear for sliding-window), MiniMaxM2-style sigmoid routing with per-layer routed scaling, and both legacy (`mlp_only_layers` + `decoder_sparse_step`) and current (`mlp_layer_types`) MoE-vs-dense schemas.
