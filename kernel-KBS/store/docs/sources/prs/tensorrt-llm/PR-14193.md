## Summary

Add SPDX-License-Identifier: Apache-2.0 headers to 95 test files under tests/unittest/auto_deploy/ and 2 YAML config files under tensorrt_llm/_torch/auto_deploy/config/ that were missing them.

These files are copied to the standalone llmc repo (NVIDIA/llm-compiler) by create_standalone_package.py, so fixing them at the source ensures the standalone repo passes OSRB license compliance review.
