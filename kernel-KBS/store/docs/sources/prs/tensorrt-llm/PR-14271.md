## Summary

This commit does the following clean ups:

* get rid of device placement directives during mistral model construction (this would lead to confusing warning messages about weight init exceptions).
* get rid of unused `cacheable` attribute and make assumptions for embedding caching support for chunked prefill more explicit in the docstring.
