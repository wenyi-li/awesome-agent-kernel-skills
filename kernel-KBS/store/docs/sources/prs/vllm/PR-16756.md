## Summary

This PR implements the fusion of fp8 quantization onto attention, described in #16220. It performs this fusion using a new `AttnFusionPass`, which uses the pattern matcher and only performs the fusion if the backend supports it. It is currently off by default, pending more robust V1 support and perf

## Problem

[torch.compile][ROCm] Fuse quantization onto attention using a torch.compile pass
