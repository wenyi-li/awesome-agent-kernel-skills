This PR fixes an incorrect LoRA B weight loading in fused_moe.py that caused vLLM LoRA outputs to diverge from PEFT/Transformers.

## Purpose

In vLLM, the LoRA B weight for MoE layers was assumed to have shape (output_dim, num_experts, rank), while PEFT defines it as (output_dim, rank, num_experts)
