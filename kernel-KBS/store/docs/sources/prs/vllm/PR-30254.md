## Purpose

#29424

This PR implements GPTQ support for MoE models with LoRA.

Previously, enabling LoRA on GPTQ-quantized MoE models would fail because `GPTQMarlinMoEMethod` did not explicitly support the modular GEMM interface required for LoRA. This PR enables the modular kernel path for GPTQ MoE
