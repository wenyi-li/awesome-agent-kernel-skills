## Purpose

Move NVFP4 GEMM management into the kernels/linear/ abstraction, matching the pattern used by FP8/Int8/MP kernels. Each backend (CUTLASS, FlashInfer, Marlin, FBGEMM, emulation) is now its own `NvFp4LinearKernel` subclass. Kernel selection goes through `init_nvfp4_linear_kernel()`.
