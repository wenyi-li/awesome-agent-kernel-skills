## Summary

## Purpose
For upcoming Nvidia releases, we require support for running non-gated activations for NVFP4 modelopt checkpoints.
This PR enables this by:
1. Modifying the sizes of the modelopt nvfp4 MoE weights according to the value of `self.moe.is_act_and_mul`
2. Calling the flashinfer cutlass fused
