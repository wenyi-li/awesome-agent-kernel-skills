## Summary

This change enables CUTLASS to utilize quantized 4‑bit weights and activations via NVIDIA’s NVFP4 format on the new Blackwell architecture (compute capability SM 120).

Tested on:
- NVIDIA RTX 5090 (Blackwell, SM120)
- All existing unit tests pass.

## Problem

Support CUTLASS NVFP4 (w4a4) for Blackwell Geforce GPUs (SM120)
