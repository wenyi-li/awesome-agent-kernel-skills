## Summary

## Motivation

NSA indexer may call `torch.cuda.mem_get_info` on the large MQA logits path. This is a host-syncing query on a latency-sensitive prefill path. Reported-by: @samuellees

## Modifications

- Cache a per-device MQA logits budget instead of querying free memory on every large prefill.
- Reuse `cu_seqlens_q` in the chunked paged path instead of allocating it inside every chunk loop.
