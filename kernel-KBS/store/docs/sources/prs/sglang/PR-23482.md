## Summary

## Motivation

Only SDPA and FIA backends available for Ascend NPU

## Modifications

Add attention backends for Laser, Block Sparse and Rain Fusion attentions.

## Accuracy Tests

Output generated with ```Laser attention``` aligns with the baseline SDPA.
Output generated with ```Block Sparse or Rain Fusion attentions``` depend on its parameters `sparsity` and `skip_first_steps`. When using such backends one has to find trade-off between speed and accuracy.
