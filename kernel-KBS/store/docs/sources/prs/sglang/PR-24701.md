## Motivation
Ref: https://github.com/sgl-project/sglang/issues/24699
Step3-VL requests with JPEG images could fail with:

```text
TypeError: cannot unpack non-iterable builtin_function_or_method object
```

The failure happens because `ImagePatcher.__call__()` executes:

```python
img_width, img_height = img.size
```

Almost the same error  with Deepseek-OCR/OCR2
