## Summary

## Summary

Fall back to the existing non-JIT masked SiLU+mul quant path when the DeepGEMM JIT EP activation kernel does not support the current `N/G` shape. Supported shapes still use the JIT activation fast path. (N/G is not multiple of 4)

## Error Reproduce

```bash
python3 -m sglang.launch_server \
  --model-path Qwen/Qwen3-30B-A3B-FP8 \
  --trust-remote-code \
  --host 0.0.0.0 \
  --port 30000 \
  --tensor-parallel-size 4 \
  --moe-a2a-backend deepep \
  --deepep-mode auto \
  --moe-runner-backend deep_gemm \
  --cuda-graph-max-bs 32 \
  --max-running-requests 32 \
  --mem-fraction-static 0.7
```
