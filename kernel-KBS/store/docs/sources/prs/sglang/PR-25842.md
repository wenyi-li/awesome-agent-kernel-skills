## Summary

- Align the SGLang diffusion benchmark/profile skill presets with `scripts/ci/utils/diffusion/comparison_configs.json` from the Nvidia nightly diffusion comparison job.
- Remove explicit steps/guidance overrides from nightly-aligned `sglang generate` presets when the nightly config omits those fields.
- Replace Wan nightly `--720p` shortcuts with explicit `--width=1280 --height=720`, add the missing LTX-2.3 `--cfg-parallel-size=2`, and update docs/recipes accordingly.
- Add `--validate-nightly-alignment` to `bench_diffusion_denoise.py` so future drift against the nightly comparison config is caught locally.

## Why

The diffusion skills had drifted from the current Nvidia nightly benchmark configuration. Some presets still encoded old/default sampling params, Wan commands used a shorthand instead of the explicit nightly size, and the LTX-2.3 TI2V preset missed the CI parallelism flag.
