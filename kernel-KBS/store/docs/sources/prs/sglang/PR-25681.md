## Summary

## Motivation

Allow custom spec algorithm to modify num_tokens_per_bs when doing TARGET_VERIFY mode.

## Modifications

* trtllm_mha_backend.py to derive num_tokens_per_bs rather than using fixed speculative_num_draft_tokens from server_args.
* Add get_num_tokens_per_bs() function to spec algorithm such that custom algorithm can override it.
* Use get_num_tokens_per_bs() function in cuda graph runner and model runner.
