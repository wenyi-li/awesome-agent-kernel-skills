## Summary

## Motivation

Enable breakable CUDA graph (BCG) support for eagle speculative decoding 

## Modifications

- **breakable_cuda_graph_runner.py**: Add draft worker support (capture_hidden_mode, spec_info with hidden states, replay_prepare for draft hidden state copy), multimodal/input_embeds support in BCG replay path
- **model_runner.py**: Add `force` parameter to `init_piecewise_cuda_graphs` so draft workers can explicitly initialize PCG/BCG after `init_lm_head`
- **eagle_worker_v2.py**: Call `init_piecewise_cuda_graphs(force=True)` when BCG is enabled
- **server_args.py**: Simplify prefill disaggregation — always disable CUDA graph for prefill mode (BCG handles it)
- **eagle_utils.py**: Add `get_draft_hidden_dim` utility for deriving draft hidden state dimensions

## Skipped files

- `http_server.py`: Warmup logging fix already applied in OSS
- `eagle_draft_extend_cuda_graph_runner.py`: Hidden dim calculation already refactored in OSS via `EagleDraftExtendInput.hidden_size_for`
