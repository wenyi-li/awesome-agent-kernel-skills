## Summary

## Motivation

This PR updates the version of `xgrammar` to `v0.2.0` to utilize the latest feature of `xgrammar`: [built-in `structural_tags`](https://xgrammar.mlc.ai/docs/tutorials/advanced_structural_tag.html) to provide structured generation for more models' tool calling.

## Modifications

This PR changes several .yml and .toml files to update the dependency on `xgrammar`. It also modifies the tool parsers. It renames the original `support_structural_tag` to `support_legacy_structural_tag`. For those parsers that support the legacy structural tags, it will not affect their logic. It provides several APIs for integration with `xgrammar`'s built-in structural_tags and completes support for the parser of `qwen3_coder`.
