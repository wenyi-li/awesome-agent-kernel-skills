## 📌 Description

`trtllm_ragged_attention_deepseek` kernel is available for GLM-5 but is currently blocked by the assertion.
Loosened assertion and added unit test to verify. Cutlass does **not** support this shape, so a naive implementation is used to test correctness.
Also added missing checks in cutlass code to error out on unsupported shapes.
