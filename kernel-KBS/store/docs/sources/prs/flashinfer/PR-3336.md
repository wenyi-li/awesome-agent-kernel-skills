## Summary

## 📌 Description

Replace the previous SM120 W4A16 MoE kernel split with a packed-route design.
The new W4A16 path groups routed token rows by expert before GEMM work, then drives the
fused FC1/FC2 kernel from packed route indices, block expert IDs, and a route count.

This removes the old static, dynamic, and micro W4A16 kernel files and adds W4A16-specific
prepared-weight, route-packing, FP4 helper, host utility, and fused-kernel modules under the
moe_w4a16_ prefix. The FlashInfer b12x MoE dispatch, workspace allocation, tracing, and tests
are updated to use the new W4A16 backend while preserving the existing caller-facing interfaces.
