## Summary

Extends the Blackwell CUTLASS Flash Attention (FMHA) implementation to support head dimension 64, broadening model compatibility beyond the initial head_dim=128. Introduces conditional tile sizing based on head dimension and achieves peak throughput of 872 TFLOPS/s on B300 hardware with the new configuration.

## Problem

The initial Blackwell CUTLASS FMHA implementation only supported head_dim=128, limiting compatibility with models that use head_dim=64 (common in smaller or older architectures). Models like GPT-2, certain Llama variants, and other architectures with 64-dimensional attention heads could not use the optimized Blackwell FMHA kernel.

## Solution / Techniques

- Conditional tile sizing based on head dimension using `std::conditional_t`
- head_dim=64: TILE_KV = 64, head_dim=128: TILE_KV = 128
- Optimal stageCountKV=1 determined through benchmarking for the 64-dimension configuration
- Comprehensive unit tests (5616 passed, 720 skipped) and benchmark validation
- Warp-specialized persistent kernel design inherited from the base FMHA implementation

## Key Code

```cuda
// Conditional tile sizing based on head dimension
// in Blackwell CUTLASS FMHA kernel
using TILE_KV = std::conditional_t<HEAD_DIM_QK == 64, _64, _128>;
// stageCountKV=1 optimal for head_dim=64 based on benchmarking
```

## Performance

Benchmarked on B300 with CUDA 13.0:

| Configuration | Peak TFLOPS/s | Conditions |
|---------------|---------------|------------|
| head_dim=64, non-causal | 872.3 | batch=2, seq_len=32768 |
| head_dim=64, causal | 758.4 | batch=1, seq_len=65536 |

Testing: 5616 passed, 720 skipped in 203.34 seconds, confirming correctness across diverse configurations.
