## Summary

## 📌 Description

Add fused DIT (Diffusion Transformer) LayerNorm CUDA kernels for video generation self-attention blocks (WAN 2.2 and similar architectures). Three fused operation modes, each combining multiple steps into a single kernel launch:

| API | Operations Fused |
|-----|-----------------|
| `fused_dit_gate_residual_layernorm_gamma_beta` | `residual_out = residual + input * (gate + gate_bias)` → LayerNorm(gamma, beta) |
| `fused_dit_gate_residual_layernorm_scale_shift` | `residual_out = residual + input * (gate + gate_bias)` → LayerNorm → `(1 + scale + scale_bias) * norm + (shift + shift_bias)` |
| `fused_dit_residual_layernorm_scale_shift` | `residual_out = residual + input` → LayerNorm → `(1 + scale + scale_bias) * norm + (shift + shift_bias)` |

Each API produces two outputs: the residual connection result (BF16) and the normalized output (BF16, NVFP4, or MXFP8). All three are exposed via `flashinfer.diffusion_ops` (like `dsv3_ops`).

### Usage

```python
from flashinfer.diffusion_ops import (
    fused_dit_gate_residual_layernorm_gamma_beta,
    fused_dit_gate_residual_layernorm_scale_shift,
    fused_dit_residual_layernorm_scale_shift,
)
