## Summary

[PDL](https://docs.nvidia.com/cuda/cuda-c-programming-guide/index.html#programmatic-dependent-launch-and-synchronization) can be used to overlap prologue and epilogue of two kernels on Hopper or later architectures. This PR adds initial PDL support for some simple kernels such as norm and activation
