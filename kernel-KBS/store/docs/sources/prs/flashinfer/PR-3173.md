## Summary

## 📌 Description

  Adds `sm_121` to `_TMEM_MAX_ALLOC_COLUMNS_MAP` in
  `flashinfer/cute_dsl/attention/compat.py`. SM120 and SM121 share the same 12.x compute capability spec, so the value (512) matches `sm_120`.

  Without this entry, `get_max_tmem_alloc_cols` raises `ValueError` on SM121 devices when `cute.arch.get_max_tmem_alloc_cols` is unavailable and the path falls through to the local map.
